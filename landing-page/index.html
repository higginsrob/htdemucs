Getting Started

Quick Start with Docker

Get started in seconds with our pre-built Docker image. Just mount your input and output folders and you're ready to go!

üìã Prerequisites:
  • Docker installed and running
  • 4GB+ RAM recommended
  • Optional: NVIDIA GPU for faster processing

Basic Usage

Basic Docker Command
# Pull the image
docker pull higginsrob/htdemucs:latest

# Run the server
docker run -d --rm \
  --name htdemucs \
  -p 8080:8080 \
  -v $(pwd)/output:/app/output \
  -v $(pwd)/models:/data/models \
  higginsrob/htdemucs:latest

Then open http://localhost:8080 in your browser!

With GPU Support (NVIDIA)

Docker with GPU
docker run -d --rm \
  --name htdemucs \
  --gpus all \
  -p 8080:8080 \
  -v $(pwd)/output:/app/output \
  -v $(pwd)/models:/data/models \
  higginsrob/htdemucs:latest

Folder Setup Explained

Input Folder Place your audio files here. Supported formats: MP3, WAV, FLAC, M4A
Output Folder Separated stems will be saved here in organized folders by model type
Models Folder AI models are cached here to avoid re-downloading on each run
